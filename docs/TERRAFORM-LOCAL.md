# Terraform pour Infrastructure Locale

## Note
Ce projet utilise Docker et Kubernetes localement.
Pour un déploiement cloud AWS, les modules Terraform sont disponibles
dans la branche `aws-deployment`.

## Infrastructure Actuelle
- Docker : Conteneurisation
- Minikube : Kubernetes local
- Jenkins : CI/CD
- SonarQube : Qualité du code
- Trivy : Sécurité
- Prometheus/Grafana : Monitoring
